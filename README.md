# alish02 - HuggingFace Space

Free AI chat interface powered by HuggingFace Inference API.

## Features
- Multiple providers support (NVIDIA NIM, OpenRouter, HuggingFace)
- Various models available
- Chat interface with history

## Environment Variables
- `HF_TOKEN` - Your HuggingFace token
- `NVIDIA_API_KEY` - NVIDIA API key (optional)
- `OPENROUTER_API_KEY` - OpenRouter API key (optional)

## Usage
1. Set your HF_TOKEN in Space secrets
2. Type a message and send
3. Chat with AI!
